      DESCRIPTION

      		 
      Purpose:

To use the GeneConnect Simple Search Interface to retrive associated genomic identifiers for a single locus ,with one/more Input Data Source and corresponding genomic indentifier's provided as input.
(Currently testing can be done only on dummy data)

Setup:

Netscape 7/Mozilla Firefox  1.5/Internet Explorer 6 must be installed .

Procedure:

1.Follow the link : http://128.252.178.209:9091/geneconnect/Home.do and click on 'Simple Search'.

2.Select Input Data Source from the 'Data Source' drop down and type in the genomic identifier corresponding to the Data Source in the area provided.

3.(i)More Input Data Source can be selected using 'Add more' button . Please note that a particular Data Source can be selected only once. e.g. Ensemble Gene cannot be selected as Input Data Source more than once.

3.(ii)Also, the 'Paste ID' button can be used to paste the genomic identifiers.The user must make sure that the genomic identifiers pasted are in correct order as indicated by the name of the Input Data Source in the corresponding drop down.

4.Select one/more Output Data Source by checking the corresponding check boxes.


Input/Data:

1.Genomic indentifier for all the data sources selected as the 'Input Data Source' from the drop down.
(Selection of atleast one Input Data Source is mandatory.)
2.Select one/more Output data Sources.

Expected Output:

1.Results page should show the following:

(i)All the genomic identifiers associated with the Input genomic identifiers must be displayed.

a.If ,for a particular Input genomic identifier there is more than one genomic identifier in a particular data source , then both the genomic identifiers must be provided in different rows.

b.If , for a particular Input genomic identifier there is no corresponding genomic identifier in the data sources selected as ouput , then the particular entry for that data source should display "No matches found ".

(iv)On checking the Frequency , a frequency value for each of the genomic identifier must be displayed.

(v)On checking the Confidence Score , a confidence score value for a particular set of genomic identifiers (per row) must be displayed.



Verification Logic:

1.The correctness of the results must be verified by searching the genomic identifiers on the website. e.g. If Entez Gene is selected as Input Data Source, then the genomic identifier provided must be searched in the NCBI-GENE, and the associated genomic identifiers can be found in 'Additional links','Related Sequences' and 'NCBI Reference Sequnces' section.
