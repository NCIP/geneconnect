      Purpose:
To ensure that the Ensembl Transcript fasta files are formatted by format-db which are later going to be used as destination database for Sequence Alignment.

Workflow details:
1.The GeneConnect Server downloads the data from the website, the Parsers parse the datafiles to create link files and  the uploader loads the data from the link files in to the respective base tables in the GeneConnect database. 2.This Parser is involved in the GeneConnect First Run ? Preparation run for preparing Blast Destination database.

Setup:
1. Verify following entries in UpdateLinux.sh:
b. The JVM parameter -DentityExpansionLimit = 5000000 has been specified in the java command.
c. The JVM parameter -Xmx512M has been specified in the java command. 
d. Check if JAVA_HOME, ORACLE_HOME, PATH AND CLASS_PATH are set properly.
e. Set the parameters of the java command:
java  -Xmx512M  -DentityExpansionLimit=5000000 -Djava.security.manager  -Djava.security.policy=java.policy com.dataminer.server.jobmanager.JobManager

2. Verify the path of the input data file (EnsemblTranscriptHumanTestFile.fa, which resides in the main folder from which GeneConnect server is run) in the config file (/Config/CommandFile.xml).

3. Check the server.properties file (in the 'Config' folder) and verify the database driver name and the dbURL and other database arguments including the TNS service name for the Oracle database, which can be any one of the following:
a) oracle.jdbc.driver.OracleDriver : for Driver name
b) jdbc:oracle:oci8:@<database name> : for database URL 

4. Check the external FormatdbEnsemblTrans.pl Perl Parser set up as follows:
Ensure that the external parser's .sh file is present in the 'ParserScripts' folder.

5. Add the following entry in Config/CommandFile.xml. This denotes that 
a) Test file is present in '/GeneConnectServer' directory of local machine. 
b) FormatdbEnsemblTrans.sh will call the external perl parser

<DataSource name="ENSEMBL_TRANSCRIPT" type="" siteurl="opt1" 
	 	username="anonymous" password=" ">

		<BaseDirectory dir="/GeneConnectServer" islocal="true">
	 		<File name="EnsemblTranscriptHumanTestFile.fa"/>
	 	</BaseDirectory>		
</DataSource> 

6. Verify that Perl, Bioperl and Blast (standalone) is installed and   paths of Perl and Bioperl are specified in the UpdateLinux.sh file.

7. Run the script UpdateLinux.sh.
